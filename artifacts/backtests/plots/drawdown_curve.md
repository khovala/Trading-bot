# Анализ графика `drawdown_curve`

## Что отображено

- Кривая `Drawdown` показывает текущую просадку стратегии относительно исторического пика капитала.
- Значение `0.0` означает новый максимум капитала, значение `0.5` — просадка 50%, и т.д.

## Ключевые числа

- Максимальная просадка (Max DD): `0.97926` (то есть ~`97.93%`).
- Момент Max DD: `2026-03-14T10:46:00+00:00`.
- Медианная просадка: `0.84980` (~`84.98%`).
- `P90` просадки: `0.97079` (~`97.08%`).
- `P99` просадки: `0.97825` (~`97.82%`).

## Наблюдения по устойчивости потерь

- Количество баров с DD >= 10%: `77,225`.
- DD >= 20%: `75,256`.
- DD >= 50%: `64,137`.
- DD >= 80%: `44,591`.
- DD >= 90%: `29,857`.
- Самый длинный непрерывный период с DD >= 50%: `64,041` бара (с `2026-01-28T23:30:00+00:00` до `2026-03-14T10:50:00+00:00`).

## Интерпретация

1. Просадка не эпизодическая, а **длительная и почти постоянная**.
2. Даже «типичное» состояние стратегии (медиана) — глубокая просадка ~85%, что критично для реального капитала.
3. Профиль кривой указывает на отсутствие эффективных механизмов контроля потерь (kill-switch/cut-risk/vol-targeting).

## Вывод

По drawdown-профилю стратегия находится в зоне неприемлемого риска. Даже при дальнейшей работе с доходностью первым приоритетом должен быть блок risk-management: ограничение позиции, ограничение оборота, стоп-условия и контроль режима рынка.
